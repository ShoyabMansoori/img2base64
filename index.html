🧭 How to Use

📤 Convert Image to Base64

  • Click on the “Choose File” button to select your image(s).
  • Click “Convert” to instantly get the Base64 output.
  • Copy the Base64 string, or download it as a .txt file for later use.
  • Optional: Preview the encoded image and check file dimensions and size.

📥 Convert Base64 to Image

  • Paste the Base64 string into the input field or upload a .txt file containing it.
  • Click “Convert” to decode the image.
  • View a live preview of the output image along with size and resolution.
  • Download the image file to your device if needed.
